## 1. Introduction

This document provides visual representations of current ("as-is") and future ("to-be") processes for the Adaptive Client Questionnaire System. These process maps illustrate the flow of activities, decision points, and interactions between users and the system.

---

## 2. Process Overview

### 2.1 Core Processes

| Process ID | Process Name | Actor(s) | Description |
|------------|--------------|----------|-------------|
| PROC-001 | User Registration | Guest → Client | Account creation flow |
| PROC-002 | Questionnaire Completion | Client | Answering questions |
| PROC-003 | Document Generation | System | Creating documents |
| PROC-004 | Document Review | Developer | Review and approval |
| PROC-005 | Document Download | Client | Accessing approved documents |
